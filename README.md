# 원티드 프리온보딩 인턴십 사전과제

- 프리온보딩 인턴십 지원을 위한 사전과제입니다.
- `create-react-app`으로 로그인/회원가입/투두리스트 기능을 제작했습니다.
- 그 외 `context-API`, `react-router-dom`, `axios`, `tailwindCSS` 를 사용했습니다.

## 로컬 실행 방법

```
$ npm i
$ npm run start
```
